# 🧮 Calculadora de IMC

Projeto simples para praticar **JavaScript** (com **HTML** e **CSS**).  
Calcula o Índice de Massa Corporal (IMC) a partir de **peso** e **altura**.

## 🚀 Tecnologias
- HTML5
- CSS3
- JavaScript (ES6+)

## ▶️ Como executar
Basta abrir o arquivo `index.html` no navegador.
